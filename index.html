想定整形地分析ツール（GeoJSON → SVG 加工ツール）

測量データの変換・回転・寸法表示をローカルで実行

ローカル動作
操作説明

このツールは 公図ビューア for 登記所備付地図データ からダウンロードした公図や測量図のGeoJSONファイルを、路線価評価用の参考資料に加工するためのツールです。

本ツールは外部との通信を必要とせず、完全にローカルで動作します。 動作に必要なコードはHTMLファイル内に組み込んであるため、当該HTMLファイル単体で動作します。

なお、公図や測量図のGeoJSONファイルは外部サイトからダウンロードして別途用意する必要があります。

1. 事前準備

  1. 公図ビューア for 登記所備付地図データ にアクセスする。
  2. 調べたい地域の県、市区町村、地番区域（大字や丁目）を指定し、表示された地番から調べたいものにアクセスする。
  3. 表示された地図からGeoJSONファイルをダウンロードしたい地番をクリックして選択する。
  4. 左側のアイコンから「GeoJSON形式保存」をクリックする。
  5. 表示されたポップアップから、対象データ＞「選択中の筆のみ」を選んで「保存」を押す。
  6. 選択した地番のGeoJSONデータがダウンロードされる。
  7. 複数の筆から成る評価単位を評価したい場合は、上記の手順（3〜6）を繰り返し、評価対象の筆を1筆ずつすべてダウンロードする。

2. ツールに読み込む

  1. このツール（想定整形地分析ツール（GeoJSON → SVG 加工ツール））を開く。
  2. GeoJSONファイルを選択ボタン、またはGeoJSONファイルを読込領域に ドラッグ＆ドロップしてアップロードする。
    • 複数筆から成る土地はまとめてアップロードしてください。
  3. 読み込んだGeoJSONファイルが表示領域にSVG画像として表示される。 縮尺はGeoJSONファイル内に記述がある場合、自動的に読み取る。

3. 想定整形地を生成する

  1. 選択モードから①辺選択②頂点選択のいずれかを選択し、次の通り正面路線に接する部分を選択する。 ここで選択した部分を基に、次のステップで想定整形地を自動的に生成する。
    1. 辺選択モードを選択した場合、読み込んだGeoJSONファイル上で頂点同士をつなぐ辺になっている部分を選択する。
    2. 頂点選択モードを選択した場合、読み込んだGeoJSONファイル上から頂点を選択する。 隣り合っていない頂点も選択することができるので、道路に接している辺が複数の頂点から成っていて細切れになっている場合や、 正面路線に面している辺が凹形になっているため辺を選択しても適切な角度が取れない場合に使用する。
      • この時点で選択された辺の想定される実寸距離が m（メートル）として表示される。
  2. 分析実行ボタンを押すと、想定整形地が表示される。 想定整形地の間口距離と奥行の長さ、現在選択されている縮尺も表示される。
    • 角度維持オプションを選択していると、想定整形地を基準として垂直な角度が維持される。
  3. A4印刷ボタンを押すと、基本的にはA4で印刷される。 A4用紙縦1枚で印刷することを想定しての機能だが、利用者の印刷設定が異なる場合はそれに従う。
    • 当該A4印刷機能は基本的にいつでも有効なため、手順2の時点で印刷することで、 選択した辺の長さを表示した状態を印刷することもできる。間口距離や「ここからここまで何m」などを示すために使うこともできる。
    • 当該A4印刷機能は正確な縮尺で印刷するものでは全くない。ツールが読み取り、表記しているｍ㍍は座標距離であるため、基本的に記載されている距離がそのまま地図上の距離である。
  4. 表示画像はSVG（ベクトル画像）であり、SVGダウンロードボタンを押すと現在表示している画像をダウンロードできる。 画像はA4縦印刷可能な縦横比で出力するように調整している。
  5. 不具合が生じた場合などは リセットボタンを押すと現在の画像の状態がリセットされる。 角度維持オプションを選択したままリセットすると現在の角度を保ったまま辺の選択などがリセットされる。
  6. 表記上の距離ｍ㍍はGeoJsonファイルをそのまま返還したSVGファイル上の座標距離を採用している。 表記上の距離に違和感がある場合や実測距離と差がある場合は縮尺を手動で変更することができる。

4. 色などを調整する

  1. スタイル設定を開くとSVG画像やページの色、SVG背景の透過具合などを調整できる。
  2. ページ右上の 寒色 暖色 オプションを選択すると背景色のテーマが切り替わる。もう一度押すと元に戻る。
